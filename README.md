# 🎓 IT105 Final Project – Student Management System

## 📖 Description

This project is a **Student Management System** developed for IT 105 – Information Management I. It aims to manage student data efficiently using a structured and normalized database design.

The system will simulate real-world academic processes such as managing student records, courses, and enrollments.

## 🎯 Project Scope

The system will:

* Store and manage student information
* Handle course and enrollment data
* Support basic database operations (CRUD)
* Ensure data integrity using relationships and constraints

## ⚙️ How to Restore and Run the System

* Clone or download the project repository
* Locate the backup folder and import the included .sql file into your database system
* Start your local server (if web-based)
* Open the project in your browser (`http://localhost/student-management-system`)


## ✨ Key Features and Functionalities

* 👤 **Student Management**

  * Add, update, delete, and view student records
  * Store personal and academic information

* 📚 **Course Management**

  * Manage course details (course name, code, units)
  * Assign courses to students

* 📝 **Enrollment System**

  * Enroll students in selected courses
  * Track student enrollments per term or semester

* 🔄 **CRUD Operations**

  * Perform Create, Read, Update, and Delete operations across all entities

* 🔗 **Relational Database Design**

  * Structured tables with proper relationships (e.g., Student–Course–Enrollment)
  * Use of primary keys and foreign keys

* 🛡️ **Data Integrity and Constraints**

  * Enforced constraints (NOT NULL, UNIQUE, FOREIGN KEY)
  * Prevention of duplicate or inconsistent data

* 🔍 **Search and Querying**

  * Retrieve specific student or course data using SQL queries
  * Filter and sort records efficiently
